# MIBTY

The website for **MIBTY NIG LTD** — a multidisciplinary technology and innovation company working across software, AI, biotechnology, nanotechnology, data, and digital platforms.

## Repository

- **Master design & architecture blueprint:** [`docs/MASTER-DESIGN.md`](./docs/MASTER-DESIGN.md) — single source of truth for brand, IA, design system, motion, tech stack, and roadmap.
- **Design tokens (machine-readable):** [`docs/design-tokens.json`](./docs/design-tokens.json) — feeds the eventual `tokens.css` + Tailwind v4 theme.

## Status

- ✅ Phase 0 — Strategy & design system (this PR)
- ⏳ Phase 1 — Marketing skeleton (Next.js 15 + Tailwind v4 + Sanity)
- ⏳ Phase 2 — Depth (full service / research / work pages)
- ⏳ Phase 3 — Signature motion & 3D
- ⏳ Phase 4 — Hardening (a11y, perf, security)
- ⏳ Phase 5 — Launch

See `docs/MASTER-DESIGN.md` §10 for the full roadmap and §13 for the launch gate.

## Stack (planned)

Next.js 15 (App Router) · React 19 · TypeScript (strict) · Tailwind CSS v4 · Radix UI · Framer Motion · GSAP ScrollTrigger · React Three Fiber · Sanity CMS · Resend · Cal.com · Plausible · Vercel.

## Branding direction

> **Engineering the next layer of reality — across code, cells, and atoms.**

Four substrates of intelligent matter — **Bits · Data · Cells · Atoms** — unified by one obsidian-and-plasma visual field. Restraint over flash. Specificity over adjectives. See `docs/MASTER-DESIGN.md` for the full system.
